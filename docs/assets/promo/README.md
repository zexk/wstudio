# wstudio promo images

Ready-to-post Patina campaign images:

- `wstudio-github-banner-1500x300.png`: compact GitHub README banner
- `wstudio-social-card-1200x630.png`: Open Graph, link previews, and social posts
- `wstudio-profile-header-1500x500.png`: profile and community headers
- `wstudio-video-banner-1280x720.png`: video thumbnails, embeds, and launch posts

The corresponding `-source.png` files preserve the larger generated originals
for future crops and exports.

Each image also has a Patina Light sibling with `-light` before the dimensions
or `-source`, for example `wstudio-social-card-light-1200x630.png`.
